📱 App de Gestión de Tareas Full-Stack

El proyecto final consiste en construir una aplicación de gestión de tareas completa usando todos los conceptos del curso.

🎯 Funcionalidades a implementar

  • Autenticación de usuarios (login / registro)
  • CRUD completo de tareas (crear, leer, actualizar, borrar)
  • Filtros por estado (pendiente, en curso, completada)
  • Sistema de prioridades (alta, media, baja)
  • Persistencia con localStorage o API externa
  • Diseño responsive con CSS Modules o Tailwind
  • Testing de componentes clave con Vitest
  • Deploy en Vercel o Netlify

🧱 Tecnologías utilizadas

React 19ViteReact Router Context APICSS ModuleslocalStorage
