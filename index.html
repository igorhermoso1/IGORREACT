📱 TaskMaster Pro — App de Tareas Full React

El proyecto final integra todos los conceptos del curso en una aplicación real y desplegable para tu portfolio.

🎯 Funcionalidades requeridas

  • Login/registro simulado con localStorage
  • CRUD completo de tareas con formulario controlado
  • Filtros por estado (pendiente/en curso/completada) y prioridad
  • Búsqueda en tiempo real con useMemo
  • Estado global con Context API o Redux Toolkit
  • Rutas protegidas con React Router
  • Diseño responsive con CSS Modules o Tailwind
  • Mínimo 3 tests con Vitest + React Testing Library
  • Deploy en Vercel o Netlify con URL pública

📅 Plan de 7 días

Día 1–2: Setup + Auth

Vite, React Router, login/registro con localStorage, rutas protegidas.

Día 3–4: CRUD tareas

Formulario controlado, lista con .map(), completar y eliminar.

Día 5: Estado global

Context API para compartir tareas sin prop drilling.

Día 6: Filtros + búsqueda

useMemo para filtrado eficiente por estado y prioridad.

Día 7: Tests + Deploy

3 tests con Vitest, build y deploy en Vercel.

🏆 Entregable: URL pública en Vercel + repositorio GitHub con README profesional.
